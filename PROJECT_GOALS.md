# GEO Analyzer — 프로젝트 개요 및 진행 현황

## 1. 프로젝트 개요

**GEO Analyzer**는 웹페이지가 **AI 검색(Generative Engine Optimization)**에 얼마나 최적화되어 있는지 분석하는 도구입니다.

- **목적**: ChatGPT, Perplexity, Google AI Overview 등이 해당 페이지를 정답 출처로 인용할 가능성을 점수화하고, 개선 방향을 제시
- **차별점**: 단순 SEO가 아니라 "AI가 이 문단을 인용할 것인가?"를 시뮬레이션

---

## 2. 핵심 기능

| 기능 | 설명 |
|------|------|
| **종합 GEO 점수** | AI 인용 가능성, 문단 품질, SEO 구조, 신뢰 신호 등 0~100점 |
| **질문 커버리지** | 실제 검색·커뮤니티 질문 vs 페이지 답변 범위 |
| **황금 문단** | AI가 인용할 확률이 높은 문단 TOP 3 |
| **개선 권고** | H2/H3 추가, FAQ 블록, 메타 태그 등 액션 플랜 |
| **PPT 리포트** | 분석 결과를 슬라이드로 다운로드 |

---

## 3. 실행 프로세스

### 3.1 전체 흐름

```
[사용자] URL 입력 → [분석] 클릭
       ↓
[API] POST /api/analyze
       ↓
[파이프라인] runAnalysis()
       ↓
[결과] AnalysisResult → UI 렌더링 (좌측 패널 + 우측 iframe)
       ↓
[옵션] PPT 리포트 다운로드
```

### 3.2 분석 파이프라인 상세

| 단계 | 처리 내용 |
|------|-----------|
| 1 | URL 정규화, 캐시 확인 (Supabase) |
| 2 | **페이지 타입 감지** — 유튜브 / 일반 웹 / 쇼핑몰 |
| 3a | **유튜브**: 메타데이터 추출 → Gemini 비디오 분석 → Tavily 질문 수집 → 결과 조립 |
| 3b | **일반 웹**: HTML 크롤링 → 메타/본문 추출 → 키워드 추출 |
| 4 | **검색 질문 수집** — Primary Topic 기반 Tavily 쿼리 → 테마 필터 → Gemini 관련성 필터 |
| 5 | **병렬 실행** — 문단 품질 분석 + Gemini 인용 채점 |
| 6 | **점수 산출** — 가변 가중치 적용, 최종 GEO 점수 계산 |
| 7 | **이슈·권고** — deriveAuditIssues + generateGeoRecommendations (Gemini) |
| 8 | AnalysisResult 반환 |

### 3.3 사용자 화면

- **좌측 패널**: 점수, 시뮬레이션, 질문 커버리지, AI 예상 질문 Top 3, 황금 문단, 발견된 이슈, PPT 다운로드
- **우측**: 사이트 프리뷰(iframe) + 이슈 마커 오버레이

---

## 4. 현재 단계

### 완료
- URL 입력 → 분석 파이프라인 (웹페이지 / 유튜브 분기)
- GEO 점수 산출 (가변 가중치, 에이스 문단 보정)
- Tavily 검색 질문 수집 → 질문 커버리지
- Gemini 기반 인용 채점, 추천/예상 질문 생성
- 이슈 마커 + 사이트 프리뷰 (iframe)
- PPT 리포트 다운로드 (실제 분석 결과 1:1 반영, 시뮬레이션 부록 옵션)
- 질문 커버리지 테마 고정 (Primary Topic 기반 쿼리·필터, 영문/한글 페이지 지원)

### 진행 중
- 점수·권고 안정화 검증
- 일부 엣지 케이스 보완

---

## 5. 최종 목표

1. **정확한 GEO 진단**: AI가 실제로 인용하는 패턴을 반영한 신뢰할 수 있는 점수
2. **실용적 개선안**: 페이지별 맞춤 H2/H3, 블록, FAQ 제안
3. **질문 커버리지 실효성**: 수집 질문이 페이지 주제와 일치 (테마 이탈 방지)
4. **PPT 리포트 활용도**: UI와 동일한 결과로 명확한 보고서 산출
5. **다양한 페이지 지원**: 일반 웹, 쇼핑몰, 유튜브 등 유형별 최적화
